DESAFIO 006

Tags em HTML

Esse exercício faz uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag utilizada para fazer isso.

PDF com as instruções do desafio 006

Tags em HTML

Em HTML 5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • título da janela
  • parágrafo
  • linha horizontal
  • imagem
  • abreviação
  • quebra de linha
  • código-fonte em uma linha
  • lista numerada
  • lista não ordenada
  • lista de definição
  • link / âncora
  • ligação de documento
  • hierarquia de títulos
  • texto em negrito
  • texto em itálico
  • marcar texto
  • texto pequeno
  • texto riscado
  • texto sublinhado
  • texto subscrito
  • texto sobrescrito
  • texto pre formatado em bloco
  • citação curta
  • citação longa em bloco
  • texto bidirecional
  • inserir estilo css
  • atrelar várias imagens
  • carregar áudio
  • visualizar vídeo
  • inserir o conteúdo de outra página
