Parágrafos

Os parágrafos em Html são textos colocados entre as tags:

<p> ... <p>

Quebra-de-linha

A quebra-de-linha produz uma quebra de linha no texto.

A tag <br> faz a quebra.

A tag <br> é uma tag vazia, ou seja, não possui tag de fechamento como a tag </p>.

Outra tag semelhante é a tag <hr> que coloca uma linha horizontal na página.

Linha horizontal

Basta colocar a tag <hr> para ela ser mostrada como as linhas abaixo.
