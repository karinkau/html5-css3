Links e âncoras

No módulo 1 - capítulo 10 - aula 1, foi ensinado a lidar com links e âncoras em Html.

Links externos: que apontam para outro local fora do site é recomendado usar os atributos:

  • target="_blank" → para abrir a página em outra aba.
  • rel="external" → para indicar aos buscadores que o link não pertence ao site pesquisado.

Links internos: são os links que formam o site pesquisado. Exemplo: essas páginas cinza, amarelo e verde são links internos do site como todas as outras páginas dos desafios.

Link para download

Para criar um link de download de um ebook ou arquivo qualquer, é preciso compactar em .zip ou .rar

Depois de compactado o arquivo, basta criar o link de download referenciando esse arquivo. Assim, se abrirá uma tela no seu sistema pedindo autorização para baixar o arquivo.

Vamos testar:

Coloquei um arquivo de texto simples, gerado no bloco de notas, com símbolos para inclusão rápida - usando apenas a tecla ALT no teclado mais números.

Exemplo: ALT + 2 = ☻

Baixe o arquivo:

Símbolos-rápidos.rar
