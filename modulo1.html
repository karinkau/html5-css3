Aprendendo HTML 5 e CSS 3

Estou aprendendo as novas tecnologias do Html 5 e Css 3 em suas versões atuais em 2020/2021, indo para 2022, pelo Curso em Vídeo de Gustavo Guanabara. Esse curso é composto por 5 módulos ao todo sendo este o primeiro módulo.

░░ Exercícios do 1º módulo:

Os exercícios propostos pelo professor, Gustavo Guanabara, são feitos durante a aula, no passo-a-passo, gradativamente, para auxiliar na fixação do conteúdo ministrado. Os códigos desses exercícios são disponibilizados no Github de Guanabara: exercicios.

É recomendado sempre a não copiar os códigos pois isso atrapalha a fixação da matéria.
